# Ecommerce Dataset Dashboard

## Setup Environment

```
git clone https://github.com/mayraaard/ProyekAnalisisData.git
cd ProyekAnalisisData
pip install -r requirements.txt
```

## Run steamlit app
```
steamlit run ./dashboard/dashboard.py
```
